iQube Repository Overview

Web3 app — NFT data containers with encryption · React, Vite, Solidity

What is iQube?

iQube is a capstone project: a Web3 app for minting and managing iQube NFTs — each NFT acts as a data container with an on-chain encryption key. Users store encrypted metadata (e.g. on IPFS), mint an NFT that points to that data and stores the decryption key on-chain (owner-only access), then can transfer or decrypt via the app.

Stack: React 18 · TypeScript · Vite · Tailwind · Solidity (ERC721) · Polygon / Avalanche · IPFS (Pinata) · Thirdweb & Metamask
